## 1. 产品目标

打造一套量化选股雷达系统，能够：
1. **多维度筛选**：基于6个核心量化指标筛选优质标的
2. **智能评分**：自动计算综合评分，输出Top榜单
3. **前瞻估值**：整合分析师预期数据，提供前瞻PE估值参考

---

## 2. 核心量化指标定义

### 2.1 指标映射关系

| 维度 | 财务报表来源 | 量化指标 | 计算公式 | 说明 |
|------|-------------|----------|----------|------|
| **高增长** | 利润表 | 营收增长率 | (本期营收-上期营收)/上期营收 × 100% | YoY同比增速 |
| **高增长** | 利润表 | 净利润增长率 | (本期净利润-上期净利润)/上期净利润 × 100% | YoY同比增速 |
| **优资产** | 资产负债表 | 资产负债率 | 负债总额/资产总额 × 100% | 越低越好，<50%为优 |
| **优资产** | 资产负债表 | 现金占比 | 货币资金/总资产 × 100% | 资产质量指标 |
| **低估值** | 行情+财务 | 前瞻PE | 当前股价 / 分析师预期EPS | 使用一致预期数据 |
| **抗通胀** | 现金流量表 | 经营现金流/净利润 | 经营活动现金流净额/净利润 | >1为优，验证盈利质量 |

### 2.2 指标数据源映射

| 指标 | 推荐数据源 | 备用数据源 | 字段示例 |
|------|-----------|-----------|----------|
| 营收增长率 | AKShare利润表 | Tushare财务指标 | `total_revenue` |
| 净利润增长率 | AKShare利润表 | Tushare财务指标 | `net_profit` |
| 资产负债率 | AKShare资产负债表 | 东财财务指标 | `total_liab/total_assets` |
| 现金占比 | AKShare资产负债表 | 东财财务指标 | `cash_equivalents/total_assets` |
| 前瞻PE | AKShare盈利预测-东财 | 同花顺盈利预测 | `forward_pe` |
| 经营现金流/净利润 | AKShare现金流量表 | Tushare现金流 | `n_cashflow_act/net_profit` |

## 3. 评分算法

### 3.1 权重分配

| 维度 | 指标 | 权重 | 评分逻辑 |
|------|------|------|----------|
| 高增长 | 营收增长率 | 15% | 标准化后加权 |
| 高增长 | 净利润增长率 | 15% | 标准化后加权 |
| 优资产 | 资产负债率 | 15% | 反向指标，越低分越高 |
| 优资产 | 现金占比 | 10% | 标准化后加权 |
| 低估值 | 前瞻PE | 25% | 反向指标，越低分越高 |
| 抗通胀 | 经营现金流/净利润 | 20% | 标准化后加权 |

**总权重 = 100%**

### 3.2 标准化方法

采用**极差标准化**（Min-Max Normalization）：

```python
# 正向指标（越大越好）
score = (value - min_value) / (max_value - min_value) * 100

# 反向指标（越小越好）
score = (max_value - value) / (max_value - min_value) * 100
```

### 3.3 综合评分公式

```
TotalScore = Σ(IndicatorScore × Weight)
           = 营收增长率分×0.15 + 净利润增长率分×0.15 
           + 资产负债率分×0.15 + 现金占比分×0.10
           + 前瞻PE分×0.25 + 现金流质量分×0.20
```

### 3.4 筛选阈值

| 指标 | 筛选条件 | 说明 |
|------|----------|------|
| 营收增长率 | > 10% | 正增长基础门槛 |
| 净利润增长率 | > 10% | 正增长基础门槛 |
| 资产负债率 | < 60% | 排除高杠杆企业 |
| 现金占比 | > 5% | 基本流动性要求 |
| 前瞻PE | < 行业平均PE × 1.2 | 相对估值合理 |
| 经营现金流/净利润 | > 0.8 | 盈利质量基础 |

---

### 3.5 行业分散度（Sector Neutrality）



为了避免选股结果过度集中于单一热门行业，引入**行业中性化**逻辑：

1. **行业分类**：采用申万一级行业分类（或同等标准的新浪/东财行业板块）。

2. **分组优选**：在计算出股票的综合评分后，按行业进行分组。

3. **名额限制**：每个核心行业内部按综合评分降序排列，仅保留前 2-3 名的优质标的进入最终精选名单。
