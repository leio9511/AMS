## 1. 产品概述

### 1.1 产品定位

AMS 是一套**全自动化的 A 股无风险套利监控系统**，覆盖三大核心场景：

| 模块 | 场景 | 频率 |
|------|------|------|
| **模块一** | ETF 折溢价套利监控 | 盘中实时 |
| **模块二** | 可转债折价套利监控 | 开盘时段 |
| **模块三** | 量化选股模型 | 每周六 |

### 1.2 核心价值

- **实时发现**: 自动监控市场，发现套利机会
- **智能分析**: 判断机会真实性，过滤虚假信号
- **操作建议**: 给出具体操作建议，降低决策成本

| 数据 | 来源 | 刷新频率 |
|------|------|----------|
| ETF 现价 (bid1/ask1) | 腾讯行情 API | 30秒 |

| 美股隔夜走势 | 新浪财经/东方财富 | 日更 |

### 2.4 监控策略

#### 2.4.1 折溢价计算

```
折溢价率 = (ask1 / IOPV - 1) × 100%

- 正值 = 溢价 (买贵了)
- 负值 = 折价 (买便宜了)
```

#### 2.4.2 异动触发条件

| 条件 | 阈值 | 推送级别 |
|------|------|----------|
| 折价 > 0.8% | 可能有套利窗口 | 🟢 关注 |
| 折价 > 1.5% | 明显套利机会 | 🟢 重点 |
| 溢价 > 3% | 溢价套利机会 | 🔴 关注 |
| 溢价 > 5% | 明显溢价套利 | 🔴 重点 |

#### 2.4.3 真实性判断逻辑 (核心)

**问题**: IOPV 有时滞，折价可能是"虚假机会"

**判断策略**:

```
┌─────────────────────────────────────────────────────────────────┐
│                    ETF 折价真实性判断流程                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  IF ETF底层资产 == 美股指数:                                     │
│      ├─ 检查隔夜美股走势                                        │
│      │   ├─ 美股隔夜下跌 > 1% → 折价反映预期，非真实机会        │
│      │   │   └─ 建议: "观望，等待净值更新"                      │
│      │   ├─ 美股隔夜上涨 → 异常折价，可能是真实机会             │
│      │   │   └─ 建议: "关注申购通道开放情况"                    │
│      │   └─ 美股休市 → 使用前一日净值，存在时滞风险             │
│      │       └─ 建议: "谨慎操作，核实净值"                      │
│      │                                                          │
│  ELIF ETF底层资产 == 港股:                                      │
│      ├─ 检查港股实时走势                                        │
│      │   ├─ 港股下跌中 → 折价反映预期                           │
│      │   └─ 港股横盘/上涨 → 可能是真实机会                      │
│      │                                                          │
│  ELIF ETF底层资产 == A股:                                       │
│      ├─ IOPV 实时性好                                           │
│      └─ 折价 > 0.5% 即可关注                                    │
│                                                                 │
│  额外检查:                                                       │
│  ├─ QDII 申购额度是否充足？                                     │
│  ├─ ETF 是否暂停申购？                                          │
│  └─ 折价是否已持续一段时间？                                     │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

#### 2.4.4 推送内容格式

**折价机会推送**:
```
🟢 【ETF 折价机会】
━━━━━━━━━━━━━━━━━━━━━━━

{操作建议}
```

## 3. 模块二：可转债折价套利监控

### 3.1 监控目标

**发现开盘时段的可转债折价套利机会**

### 3.2 监控时段

| 时段 | 说明 |
|------|------|
| **09:25-09:45** | 核心监控时段 (开盘20分钟) |
| 其他时段 | 不监控 |

### 3.3 筛选条件

| 条件 | 阈值 | 说明 |
|------|------|------|
| **溢价率** | < 5% | 低溢价才有套利空间 |
| **溢价率** | < 0% (折价) | 最佳机会 |
| **剩余规模** | < 3亿 | 小盘转债弹性大 |
| **成交活跃** | 成交额 > 1000万 | 流动性保证 |

### 3.4 操作建议逻辑

```
┌─────────────────────────────────────────────────────────────────┐
│                    可转债操作建议逻辑                            │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  IF 溢价率 < 0% (折价):                                         │
│      ├─ 检查正股走势                                            │
│      │   ├─ 正股上涨 → 转债补涨预期                             │
│      │   │   └─ 建议: "买入转债，等待补涨"                      │
│      │   └─ 正股下跌 → 转债有债底保护                           │
│      │       └─ 建议: "买入转债，下跌空间有限"                  │
│      │                                                          │
│  ELIF 溢价率 0% ~ 3%:                                           │
│      ├─ 检查正股弹性                                            │
│      │   ├─ 正股高波动 → 转股价值提升空间大                     │
│      │   │   └─ 建议: "关注正股走势，择机介入"                  │
│      │   └─ 正股低波动 → 套利空间有限                           │
│      │       └─ 建议: "观望"                                    │
│      │                                                          │
│  ELIF 溢价率 3% ~ 5%:                                           │
│      ├─ 需要正股上涨才能获利                                    │
│      └─ 建议: "溢价偏高，需正股配合"                            │
│                                                                 │
│  额外检查:                                                       │
│  ├─ 转债是否临近强赎？(溢价归零风险)                            │
│  ├─ 转债是否临近回售？(下修博弈机会)                            │
│  └─ 正股是否公告重大事项？                                      │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 4. 模块三：量化选股模型

### 4.1 监控目标

**多维度筛选优质标的，周六推送周报**

### 4.2 运行频率

| 时间 | 操作 |
|------|------|
| **每周六 10:00** | 运行选股模型，推送周报 |

### 4.3 量化指标体系

| 维度 | 指标 | 权重 | 说明 |
|------|------|------|------|
| **高增长** | 营收增长率 (YoY) | 15% | > 10% 为优 |
| **高增长** | 净利润增长率 (YoY) | 15% | > 10% 为优 |
| **优资产** | 资产负债率 | 15% | < 50% 为优 |
| **优资产** | 现金占比 | 10% | > 10% 为优 |
| **低估值** | 前瞻PE | 25% | < 行业平均为优 |
| **抗通胀** | 经营现金流/净利润 | 20% | > 1 为优 |

### 4.4 筛选流程

```
┌─────────────────────────────────────────────────────────────────┐
│                    量化选股筛选流程                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  Step 1: 全市场扫描                                             │
│  ├─ 获取 A 股全部股票财务数据                                   │
│  └─ 排除 ST、退市风险股票                                       │
│                                                                 │
│  Step 2: 基础筛选                                               │
│  ├─ 营收增长率 > 10%                                            │
│  ├─ 净利润增长率 > 10%                                          │
│  ├─ 资产负债率 < 60%                                            │
│  └─ 前瞻PE < 行业平均 × 1.2                                     │
│                                                                 │
│  Step 3: 综合评分                                               │
│  ├─ 每个指标标准化 (Min-Max)                                    │
│  ├─ 加权计算综合分                                              │
│  └─ 按综合分降序排列                                            │
│                                                                 │
│  Step 4: 行业分散                                               │
│  ├─ 按申万一级行业分组                                          │
│  ├─ 每个行业最多保留 2-3 只                                     │
│  └─ 避免过度集中于热门行业                                      │
│                                                                 │
│  Step 5: 输出 Top 20                                            │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 4.5 推送内容格式

```
🎯 【量化选股雷达】周报
📅 筛选时间: {YYYY-MM-DD}
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

| 排名 | 代码 | 名称 | 综合分 | 增长分 | 估值分 |
|------|------|------|--------|--------|--------|
{STOCK_ROWS}

📊 【本周亮点】
{TOP_PICKS}

💡 【操作建议】
{ADVICE}

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
⚡ 数据来源: AKShare + 东财盈利预测
```

### 4.6 数据源

| 数据 | 来源 | 刷新频率 |
|------|------|----------|
